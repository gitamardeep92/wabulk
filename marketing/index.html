FAQ

Common questions

Do I need Meta Business or BSP approval?

No. WaBulk works by connecting your existing WhatsApp number via QR scan — the same way WhatsApp Web works. No Meta approvals, no BSP partnership, no verification process required.

What kinds of messages can I send?

WaBulk is designed for informational and transactional notifications — order confirmations, membership reminders, payment receipts, appointment alerts. Promotional or spam content is not permitted.

Will my WhatsApp account get banned?

We minimize this risk by enforcing humanized delays between messages (2–8 seconds), preventing duplicate sends, and limiting daily volumes. Accounts sending legitimate content to consenting users are very safe.

How do dynamic variables work?

Use {{variable_name}} in your template. Pass the values per-recipient in a vars object when calling the API. Every recipient gets a fully personalized message.

Can I use this from any programming language?

Yes — WaBulk is a standard REST API with JSON. Works with Node.js, Python, PHP, Ruby, Go, and no-code tools like Zapier and Make.

What happens when I hit my monthly limit?

The API returns a clear error. You can upgrade your plan at any time and the new limit applies immediately — no waiting for the next billing cycle.
